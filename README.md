# React Catalog with Cart

Мини-приложение на React для просмотра каталога товаров с корзиной.  

## Функционал

- Каталог товаров с карточками  
- Фильтр по категориям и поиск с debounce  
- Сортировка товаров (по цене и названию)  
- Пагинация (12 товаров на странице)  
- Корзина:
  - Добавление/удаление товаров
  - Изменение количества
  - Подсчет общей стоимости
  - Сохранение в localStorage
- UI/UX:
  - Skeleton loader при загрузке
  - Toast-уведомления
  - Адаптивная верстка (mobile/desktop)
  - Темная тема

## Технологии

- **React**
- **TypeScript**  
- **Redux Toolkit** 
- **CSS Modules**  
- **React Router** 
- **FakeStore API** 
- **Lodash**   
- **Vitest + Testing Library** 


## Установка и запуск

1. Клонируем репозиторий:

```bash
git clone https://github.com/<YOUR_USERNAME>/react-catalog.git
cd react-catalog
```
2. Устанавливаем зависимости:

```bash
npm install
```
3. Создаем файл .env в корне проекта

4. Запускаем приложение:

```bash
npm run dev
``` 
5. Открываем [http://localhost:5173] в браузере.

6. Запускаем тесты:

```bash
npx vitest
```

## Описание решения

	1.	React + TypeScript - компонентный UI, строгая типизация, предотвращение ошибок.
	2.	Redux Toolkit — управление состоянием корзины и синхронизация с localStorage.
	3.	CSS Modules — изоляция стилей и удобство адаптивной верстки.
	4.	React Router — маршрутизация между страницами каталога и корзины.
	5.	FakeStore API — быстрый доступ к JSON данным для тестирования функционала.
	6.	Lodash (debounce) — оптимизация поиска по названию товаров.
	7.	Skeleton loader и Toast — улучшение UX при загрузке и действиях пользователя.
	8.	Адаптивная верстка — корректное отображение на mobile, tablet и desktop.
	9.	Темная тема — CSS-переменные + контекст React для переключения темы.
	10.	Unit-тесты (Vitest + Testing Library) — проверка Redux и UI компонентов.
	11.	Пагинация, сортировка, фильтры — улучшение UX и удобство навигации.
